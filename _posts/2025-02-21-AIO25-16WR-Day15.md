## Correlation (Tương quan giữa các biến số)

Correlation là một phép đo thống kê cho biết mức độ và hướng quan hệ giữa hai biến số. Khi hai biến số có mối quan hệ tương quan, giá trị của một biến có thể ảnh hưởng đến giá trị của biến còn lại.

Giả sử ta có hai giá trị `x` và `y`. Khi xét về sự tương quan giữa hai biến số, ta có 3 hướng kết luận:
- Positive Correlation (Tương quan dương): Khi giá trị x tăng thì giá trị của y cũng sẽ tăng.
- Negative Correlation (Tương quan âm): Khi giá trị x tăng thì giá trị của y giảm
- No Correlation (Không tương quan): Không có mối quan hệ rõ ràng giữa x và y

### Correlation Coefficient (Hệ số tương quan)

Hệ số tương quan là một giá trị trong khoảng $[-1, 1]$, giá trị này thể hiện mức độ của mối quan hệ tương quan:
- 1: Mối quan hệ tương quan hoàn hảo dương
- 0: Không có mối quan hệ tương quan
- -1: Mối quan hệ tương quan hoàn hảo âm

### Công thức Pearson Correlation Coefficient - Tính hệ số tương quan giữa x và y

$$ r = \frac{ \sum (X - \bar{X}) (Y - \bar{Y}) }{ \sqrt{\sum(X - \bar{X})^2 \sum(Y - \bar{Y})^2} }$$

Với:
- $X$ và $Y$ là hai biến số
- $\bar{X}$ và $\bar{Y}$ là giá trị trung bình của X và Y

Hoàn thành code tính correlation, tính trung bình của x và y rồi sau đó xác định tử số và mẫu số của hàm correlation

$$ r = \frac{ \sum (X - \bar{X}) (Y - \bar{Y}) }{ \sqrt{\sum(X - \bar{X})^2 \sum(Y - \bar{Y})^2} }$$

{% highlight python %}
def correlation(x, y):
  tb_x = np.mean(x)
  tb_y = np.mean(y)

  tuso = np.sum((x - tb_x)*(y - tb_y))
  mauso = np.sqrt( np.sum((x - tb_x)**2) * np.sum((y - tb_y)**2) )
  
  if mauso == 0:
    return 0
  return tuso/mauso
{% endhighlight %}

Thư viện numpy cũng có tích hợp sẵn công cụ để tính correlation, để sử dụng công cụ này, ta có thể dùng hàm np.corrcoef()

{% highlight python %}
def correlation(x, y):
  return np.corrcoef(x, y)
{% endhighlight %}

# Cách 1 - Tính thủ công từng cặp feature

# Load dữ liệu vào pandas dataframe
